						Ejercicio 2: Lista de Compras

Descripcion

El siguiente programa en python desarrolla unas listas de compras que confirman su existencia dentro del codigo fuente del programa,
para ello, primero inicializa el inventario con unos articulos predeterminados, luego pide al usuario ingresar la cantidad de 
productos a llevar, despues el nombre del producto y finalmente la cantidad de dicho producto, y asi hasta finalizar con el numero
de cantidad de productos dicha anteriormente. 

Finalmente, el programa compara la lista predeterminados con la lista ingresada por el 
usuario para buscar los nombres dados de articulos, para buscar su precio y multilplicarlo por la cantidad ingresada por el usuario.

En el caso de que, el nombre ingresado por el usuario no coincida con la registrada en el codigo fuente, este omitara esta entrada
y pasara a la siguiente. Si todos los nombres coinciden con lo registrado en el codigo fuente, este registraesta entrada, 
para finalmente emitir un resumen de compra final con el total de todos los productos ingresados

Funcionamiento

Ejemplo 1:
                Compra Supermercado

Ingrese cantidad de productos a comprar: 1
producto 1: queso
        cantidad: 2

                        Resumen de Compra
        Cant            Nombre          Presentacion            Marca           Subtotal
        2               queso           en crema                kraft           650

        Total: 650
